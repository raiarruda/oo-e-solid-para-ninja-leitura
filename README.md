# 📚 OO e SOLID para Ninjas — Leitura  + Implementação Prática

> Estudo aplicado do livro *"Orientação a Objetos e SOLID para Ninjas"* de Mauricio Aniche, com aplicação prática em C# (.NET 8+) e Vue.js 3.
>
> GitHub do autor: https://github.com/mauricioaniche
---

## 🎯 Objetivo

- Consolidar conceitos de **OO e SOLID**
- Aplicar em cenários reais 
- Evoluir código pensando em **manutenibilidade, testabilidade e baixo acoplamento**

---

## 🧠 Abordagem

Cada capítulo contém:

- 📌 Resumo direto ao ponto  
- 🚨 Problema que resolve  
- 🏗 Aplicação prática  
- 🧪 Exemplos de código:
  - Backend (C# .NET 8+)
  - Frontend (Vue 3 + Composition API)
